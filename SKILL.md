# Site Text Audit

## Назначение
Проверяет сайт на орфографические, грамматические, логические и стилистические ошибки.

## Когда использовать
Если пользователь просит:
- проверить сайт на ошибки
- сделать аудит сайта
- проверить орфографию сайта
- проверить грамматику сайта
- найти ошибки на сайте

## Команда запуска
Всегда запускай через exec:

python3 ~/.openclaw/workspace/skills/site_text_audit/task.py "URL" PAGES

Где:
- URL — сайт пользователя
- PAGES — количество страниц для проверки

## Пример
Пользователь:
проверь сайт https://www.qifa.ru/ на 5 страниц

Команда:
python3 ~/.openclaw/workspace/skills/site_text_audit/task.py "https://www.qifa.ru/" 5

## Ответ пользователю
После запуска сообщи:
- сколько ошибок найдено
- путь к Excel-файлу
